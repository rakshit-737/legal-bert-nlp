Get started

Deploy in minutes

Run locally or deploy to Streamlit Cloud for a shareable public URL—free forever.

🖥️ Run locally

# 1. Clone the repo
git clone https://github.com/rakshit-737/legal-bert-nlp.git
cd legal-bert-nlp

# 2. Install dependencies
pip install -r requirements.txt

# 3. Launch the app
streamlit run app/streamlit_app.py
💡 Python 3.9+ recommended. GPU optional but speeds up inference significantly.

☁️ Deploy to Streamlit Cloud

  1. Fork or push this repo to your GitHub account.
  2. Go to share.streamlit.io and sign in with GitHub.
  3. Click "Deploy an app", select this repo, set main file to app/streamlit_app.py.
  4. Click "Deploy!" and wait ~2 min. Your public URL will appear.
  5. Update the app URL in docs/index.html and README.md.
🔒 No secrets required for the base demo. All ML runs in the Streamlit Cloud container.
